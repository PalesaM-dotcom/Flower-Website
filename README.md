# Pretty Blooms Website 🌸

**Pretty Blooms** is a modern, responsive flower shop website built with **HTML, CSS, and JavaScript**.  
It allows users to browse flowers, add products to a cart, and complete a checkout with billing and card details.

---

## Features

- Responsive product catalog with images and prices  
- Add to cart functionality with quantity and subtotal calculation  
- Dynamic checkout page with billing information and card details  
- Thank you message after placing an order  
- Fully responsive design for desktop and mobile devices
